DBAdmin CLI

Consola unificada para gestionar bases de datos relacionales y NoSQL. Rápida, portable y profesional.

⬇ Descargar DBAdmin.exe

📖 Manual de Uso

Al iniciar DBAdmin.exe, un menú te permite elegir el modo de trabajo:

[1] Modo Relacional   [2] Modo NoSQL

El prompt cambia a dbcli-rel > o dbcli-nosql > según corresponda.

🗄️ Modo Relacional

Soporta SQLite, PostgreSQL y MySQL.

🔌 Conexiones

# Conexión a SQLite (archivo local)
dbcli-rel > connect sqlite test.db

# Conexión a PostgreSQL
dbcli-rel > connect postgres mi_db usuario pass localhost 5432

# Conexión a MySQL
dbcli-rel > connect mysql mi_db usuario pass localhost 3306

📊 Consultas CRUD

# Seleccionar registros
dbcli-rel [sqlite: test.db] > select * from usuarios where edad > 18

# Insertar un nuevo registro
dbcli-rel [sqlite: test.db] > insert into usuarios (nombre, edad) values ('Ana', 30)

# Actualizar registros
dbcli-rel [sqlite: test.db] > update usuarios set edad = 31 where nombre = 'Ana'

# Eliminar registros
dbcli-rel [sqlite: test.db] > delete from usuarios where nombre = 'Ana'

🏗️ Estructura (DDL)

# Crear tabla
dbcli-rel [sqlite: test.db] > create table clientes (id int, nombre text)

# Eliminar tabla
dbcli-rel [sqlite: test.db] > drop table clientes

# Listar todas las tablas
dbcli-rel [sqlite: test.db] > show tables

🛠️ Utilidades

# Exportar último SELECT a CSV
dbcli-rel [sqlite: test.db] > export reporte.csv

# Ver estado de la conexión
dbcli-rel [sqlite: test.db] > status

# Cerrar conexión actual
dbcli-rel [sqlite: test.db] > disconnect

# Salir de la aplicación
dbcli-rel [sqlite: test.db] > exit

🧩 Modo NoSQL

Soporta MongoDB, Redis y Cassandra.

🍃 MongoDB (Documentos)

# Conexión a MongoDB
dbcli-nosql > connect mongodb mi_base localhost 27017

# Insertar un documento (formato JSON simplificado)
dbcli-nosql [mongodb: mi_base] > insert usuarios {"nombre": "Carlos", "rol": "admin"}

# Buscar documentos
dbcli-nosql [mongodb: mi_base] > find usuarios {"rol": "admin"}

# Actualizar documentos
dbcli-nosql [mongodb: mi_base] > update usuarios {"nombre": "Carlos"} {"rol": "superadmin"}

# Eliminar documentos
dbcli-nosql [mongodb: mi_base] > delete usuarios {"nombre": "Carlos"}

# Listar colecciones
dbcli-nosql [mongodb: mi_base] > show collections

🔴 Redis (Clave-Valor)

# Conexión a Redis (base de datos 0)
dbcli-nosql > connect redis 0 localhost 6379

# Establecer una clave
dbcli-nosql [redis: DB:0] > set mensaje "Hola Mundo"

# Obtener una clave
dbcli-nosql [redis: DB:0] > get mensaje

# Listar todas las claves
dbcli-nosql [redis: DB:0] > keys *

# Eliminar una clave
dbcli-nosql [redis: DB:0] > del mensaje

🔵 Cassandra (Columnas / CQL)

# Conexión a Cassandra
dbcli-nosql > connect cassandra mi_keyspace localhost

# Consultar registros
dbcli-nosql [cassandra: KS:mi_keyspace] > select * from registros

# Insertar un registro
dbcli-nosql [cassandra: KS:mi_keyspace] > insert into registros (id, valor) values (1, 'dato')

# Listar tablas del keyspace
dbcli-nosql [cassandra: KS:mi_keyspace] > show tables
